# SSPNet graph visualization

Simple live sensor graph with Flask and SocketIO

Getting Started
---------------

Install Python pacakages
^^^^^^^^^^^^^^^^^^^^^^^^

``bash
$ pip install -r requirements.txt
``

Run server
^^^^^^^^^^

```bash
$ python server.py
```

Open `localhost:5000` and click the **start** button

Try it out
^^^^^^^^^^

```bash
$ python client_test.py
```
